# Supplementary Materials for DTRF Emotional TTS

This repository provides supplementary materials for the paper:

**Dual-Track Residual Framework for Continuously Controllable Emotional Speech Synthesis**

The repository includes:
- Dataset split statistics
- Filelist construction descriptions
- Experimental configuration summaries
- Aggregate objective and subjective evaluation results
- Metric definitions

This repository is intended to support the data availability statement of the paper. It provides aggregate experimental materials rather than a full code release.

This repository does **not** include:
- Original ESD or VCTK audio recordings
- Generated audio samples
- Model checkpoints
- Third-party pretrained models
- Full training or inference code
- Participant-level listening-test records

The public datasets used in the paper should be obtained from their original providers under their respective licenses.
